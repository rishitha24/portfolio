Projects



1. Smart Green House

The aim of this project is to monitor and control a green house with sensors and actuators through a mobile app connected to the internet dynamically.

Sensors that were used are soil moisture, Temperature and Humidity - DHT 11 and LDR for light sensor.

Data was collected from the sensors through a microcontroller Arduino here, and stored in the firebase.

The web application and the mobile app was built using HTML5, CSS and very few bootstrap features!

Github link for the project : Click here!



2. Multiple Motor Synchronization

This project is an idea to replace huge mechanical devices like gears with an embedded device (tried with Arduino) to synchronize many motors in industries.

Using an embedded device would be benificial because we can over came wear & tear of machines also labour work can be reduced.

The technique that was used to synchronize motors was "Adjacent cross-coupling" keeping delay in the mind.



3. Spatial Modulation

This project is an idea to simulate Spatial Modualtion technique with Space Time Block Coding (STBC) to achieve higher data rate, spectral efficiency and reliability.

STBC is chosen over Spatial Multiplexing (SM) because, SM using V-Blast requires complex decoders at the receivers as they are prone to high channel-interface.
